# 🚀 Turbo Engine

**The productivity booster for Windows that works right where you need it: on the active window.**

Turbo Engine is a micro‑utility that constantly monitors the foreground window and automatically increases the CPU priority of its corresponding process.  
This gives the application you’re working with **more system attention**, resulting in a snappier, more responsive experience – without any hassle.

### ✨ How it works
- Detects the active window every 800 ms.
- Ignores system processes (explorer, svchost, etc.).
- Applies **AboveNormal** priority only to user applications that are still at Normal priority.
- Everything runs in the background, no administrator rights needed.

### 🖥️ Interface
A tiny, always‑on‑top window with just two buttons:
- **ON** – activates the turbo.
- **OFF** – turns it off.

Perfect to keep in a corner of the screen for instant control.

### 📌 Why use it?
- **Better responsiveness** in heavy apps (editors, IDEs, browsers, games).
- **Lightweight** and zero impact when off.
- Compatible with **Windows 7 through Windows 11**.
- **Open source** and straightforward (PowerShell + .NET).

### 🚀 Installation
1. Download `TurboGui.ps1` and `TurboEngine.bat`.
2. Place them in the same folder.
3. Run `TurboEngine.bat`.
4. Choose the language and click **ON**.

### 🧪 Requirements
- Windows 7/8/8.1/10/11
- PowerShell 5.1 (included by default in Windows 10+; on Windows 7 install [WMF 5.1](https://www.microsoft.com/en-us/download/details.aspx?id=54616))

O Turbo Engine é uma micro‑utilidade que monitoriza constantemente a janela em primeiro plano e aumenta automaticamente a prioridade de CPU do processo correspondente.  
Desta forma, a aplicação com que estás a trabalhar recebe **mais atenção do sistema**, resultando numa experiência mais fluida e responsiva, sem complicações.

Uma janela minúscula e sempre visível com dois botões:
- **ON** – ativa o turbo.
